Tentang Proyek EKG

Proyek ini dirancang untuk memantau sinyal listrik jantung (Elektrokardiogram) secara real-time menggunakan teknologi IoT.

Cara Kerja:

  1. Sensor EKG: Menangkap sinyal listrik dari tubuh.
  2. ESP32: Memproses sinyal analog menjadi data digital.
  3. Flask Server: Menerima data melalui internet dan menyimpannya ke database.
  4. Web Dashboard: Menampilkan grafik jantung yang bisa diakses di mana saja.

Teknologi yang Digunakan:

Python (Flask) • HTML • CSS • ESP32
